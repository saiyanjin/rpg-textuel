# rpg-textuel
projet rpg textuel dans la console en java

lancez le fichier Jeu.java pour commencer à jouer

déroulement du jeu :

le joueur choisit un pseudo puis une caste parmi : Sorcier, Elfe et Guerrier
la map apparaît
le joueur dispose alors de 3 choix :
  1. Acheter une arme
  2. Voir l'inventaire
  3. Se déplacer
le joueur choisit l'action en écrivant dans la console 1, 2 ou 3
